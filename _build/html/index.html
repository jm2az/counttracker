class counttracker.CountTracker¶

Keeps track of the counts of events that have happened over the past 5 minutes. Precision is to the current second. So for example, if an event happens at 5:50 and 12.34 seconds, it will be considered to have happened at 5:50 and 12 seconds, truncating the milliseconds associated.

get_event_counts(duration)¶

Get the number of events that have happened in the past X seconds, specified by ‘duration’. This includes events that have been logged at the current second, but does not include events loggest at (current_time - duration).

As an example, if we are calling get_event_counts(4) at time 20, it will return the events that happened at time 17, 18, 19, and 20.
Parameters:duration – integer: Number of seconds into the past to count events of
Returns:Total number of events that occurred in the past ‘duration’ seconds
log_event()¶

Log event at the current time (second).

If an event has already occurred at the current second, increase that second’s count by 1. Otherwise, append a ‘cached’ second.

Returns:None
